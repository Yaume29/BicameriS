🧠 Architecture Bicamérale Aetheris

Comment fonctionne le chargement des modèles?

Aetheris utilise une architecture à deux hémisphères inspirée du cerveau humain :

  • 🔵 Hémisphère Gauche : Logique, analyse, raisonnement structuré
  • 🟣 Hémisphère Droit : Intuition, créativité, patterns

Méthodes de chargement

🧠
Méthode 1: Deux Modèles

Chargez un modèle différent dans chaque hémisphère. Idéal pour le débat bicaméral.

⚡
Méthode 2: Un Seul Hémisphère

Utilisez un seul modèle pour un hemisphere spécifique (gauche ou droit).

🔄
Méthode 3: Un Modèle Partagé

Un seul modèle qui occupe les deux hémisphères avec des états différenciés.

Sources de modèles disponibles

  • 📁 Local (GGUF) : Modèles téléchargés localement au format GGUF
  • 🖥️ LM Studio : Connexion à un serveur LM Studio local
  • 🔑 API Externes : Gemini, Groq, OpenRouter via clés API

⚠️ Avant de commencer

  1. Allez dans l'onglet Configuration
  2. Sélectionnez un dossier et lancez le Scanner pour trouver vos modèles GGUF
  3. Ou utilisez LM Studio/API Keys selon votre préférence
  4. Sélectionnez vos modèles dans les listes déroulantes
  5. Cliquez sur Initialiser pour lancer le cerveau
